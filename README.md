### Perfil Profesional

Soy estudiante de Ingeniería de Software enfocado en la construcción de sistemas robustos y de alto rendimiento. Mi interés principal radica en la arquitectura de software, aplicando patrones de diseño (MVC, arquitectura limpia) y estructurando soluciones modulares. Más allá de usar frameworks, busco entender el funcionamiento interno de las herramientas y optimizar despliegues en entornos Linux y contenedores.

* **Especialización:** Programación Web orientada al entorno empresarial, desarrollo de ecosistemas complejos (ERP/CRM) y plataformas educativas.
* **Stack Principal:** C# (.NET) y NestJS para backend, con un fuerte enfoque en bases de datos relacionales e infraestructura.
